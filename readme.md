## Project Overview
This repository serves as my Digital Literacy Portfolio, created for my role as a Student Digital Ambassador. The purpose of this project is to help peers navigate the digital world safely and responsibly. This single GitHub repository is the main deliverable and contains all my work for the five tasks, which map directly to the five modules of the course.
